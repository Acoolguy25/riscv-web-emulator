Among OS Shell Help

Your prompt appears as <cwd>$ (current working directory).
Press Ctrl + C (Control + C) at any time to terminate all foreground processes.

Quick Start

  1. Enter a command, then press Enter.
  2. Use piping and redirection with Unix-standard operators:
    • Pipe commands: cmd1 | cmd2
    • Input redirection: cmd < file
    • Output redirection: cmd > file
  3. Command history navigation: Use ↑ and ↓ arrow keys to cycle through previously entered commands.

Supported Commands

  • cat
  • cd
  • cp
  • date
  • diff
  • edit
  • grep
  • ls
  • mv
  • rm
  • spawn
  • time
  • touch
  • run

Additional shell commands:

  • help – Displays an existential “Nobody is going to save you” message.
  • commands – Lists all available commands.
  • games – Lists available games (see below).
  • clear – Clears the terminal screen.
  • exit – Exits the shell.
  • info – Kernel and system information.

Piping and Command Chaining

The shell supports Unix-standard piping and file redirection operators |, <, and > for chaining commands and I/O:

# Count text files in directory
ls | grep ".txt" | wc

# Sort unique lines in a file
cat small.txt | sort | uniq

# Redirect output to a file
cat medium.txt > out.txt

# Use file as input
grep bee < bee_movie.txt

Process Control

  • Forking: Each command runs in its own isolated process with kernel memory protection.
  • Interrupting: Pressing Ctrl + C terminates all running processes immediately.
  • The shell waits for all processes in a pipeline to complete before returning to the prompt.

Device and I/O Handling

Standard input/output are mapped to UART 0 by default. You can redirect streams to files or other devices using pipes or redirection:

# Save directory listing to a file
ls > listing.txt

# Run commands from a script file
run < script.sh

Available Games

Launch games directly by typing their names:

  • zork – Text-based adventure
  • rogue – Dungeon crawler (h j k l to move, f to fight, Ctrl + S to save)
  • pacman – Classic arcade game (WASD to move)
  • applewars – Collect apples while avoiding enemies (WASD + Space to fire)
  • invaders – Space invaders clone (a/d to move, Space to shoot)

Screen and Kernel Limits

  • Kernel-supported screen size: 32 rows × 90 columns.
  • Maximum 10 executables and 10 arguments per command pipeline.
