RISC-V Emulator Overview

This emulator is a WebAssembly-based RISC-V RV64 platform that I developed to run a monolithic kernel and user applications directly in a modern browser. It builds upon the Simmerv project by Tommy Thorn, which itself is a fork of Takahiro’s riscv-rust repository.

Key Enhancements and Features I Added

  • Multiple UART screens: I expanded support to three UART terminals, enabling simultaneous multi-screen output instead of just one.
  • Kernel printing support: I added direct kernel log output visible in the browser’s developer console for easier debugging.
  • Real-Time Clock (RTC): I integrated an RTC device emulation to provide current time to applications.
  • Small patches: I applied various bug fixes and incremental improvements to enhance stability and usability.
  • Legacy compatibility: Throughout, I maintained core design and performance optimizations inherited from the original projects.

Technical Highlights

  • Runs fully inside the browser using WebAssembly, delivering near-native performance.
  • Implements multiple MMIO devices like UART, CLINT, PLIC, VirtIO block device, RNG, and RTC.
  • Supports Sv39 virtual memory with full paging and protection for kernel and user processes.
  • Designed for smooth integration with the Rust-based kernel and user programs I developed.
